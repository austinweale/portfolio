Geographic Listening Patterns

This website displays data about bands and where they are located

The page uses multiple APIs to get information about musicians. It can connect to a Last.fm username to display where their top artists are from. This feature uses to Last.fm API to get the user data, and the musicbrainz API to get artist locations. The other feature connects to the spotify API so you can search keywords to see in which locations these keywords are used the most in song names or band names. This also uses the musicbrainz API to get the location of the artist
